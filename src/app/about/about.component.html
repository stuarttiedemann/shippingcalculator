Shipping Calculator

An Angular 4 application

Welcome to my shipping calculator built using Angular 4. This application demonstrates some of my skills with the latest and greatest version of Angular.


Skills Used

  • Routing with Angular Router
  • API's: three API's are used in this app.
  • Services: various services are used for handling things like API calls and the passing of data between elements.
  • Promises: promises are used since the results of the google zip code search must be received before making a call to the shipping API.
  • Bootstrap 4 was used to keep the styling simple and clean.
